# BC-Tools 学生管理系统 - 操作说明书

> **版本**：v1.1.0  
> **适用对象**：学校管理员、教务处、任课教师  
> **系统目标**：致力于提供一站式的数字化校园管理方案，涵盖学生档案、成绩分析及日常教务协作。

---

## 1. 系统概览

BC-Tools 是一个现代化的 Web 学生管理平台，采用响应式设计，支持桌面端与平板端访问。本系统集成了学生生命周期管理、多维度成绩分析以及灵活的角色权限控制。

### 核心功能预览
- 📊 **多维仪表盘**：全校数据概览，实时统计学生、教师及教学动态。
- 👥 **学生档案管理**：支持批量导入、关键词检索、学籍状态全程追踪。
- 📝 **成绩与学情分析**：不仅记录分数，更提供趋势分析、班级对比及薄弱项预警。
- 📚 **作业与考试管理**：数字化记录作业完成情况，管理各类考试安排。

## 2. 快速上手

### 2.1 登录与退出
1. **访问系统**：在浏览器中输入系统地址。
2. **账号登录**：输入管理员分配的邮箱地址与密码。
3. **安全建议**：初次登录后，建议前往“系统设置 -> 个人资料”修改初始密码。
4. **退出登录**：点击页面右上角头像，从下拉菜单中选择“退出登录”。

### 2.2 界面导航
- **左侧侧边栏**：所有功能模块的入口。
- **顶部标题栏**：显示当前页面名称，并包含用户头像及通知入口。
- **收缩侧边栏**：小屏幕或需要更大操作空间时，可点击菜单图标收缩导航。

## 3. 功能模块操作指南

### 3.1 仪表盘 (Dashboard)
系统首页，提供关键指标的实时统计。
- **统计卡片**：查看当前在校学生总数、班级数量、教师总数等。
- **最近动态**：展示系统内最新发生的操作记录，如“新增学生”、“成绩发布”。

### 3.2 学生管理 (Core)
这是系统的核心模块，用于维护全校学生档案。
- **学生列表**：
    - **搜索与筛选**：支持按姓名/学号搜索，或按年级、班级筛选。
    - **查看详情**：点击操作栏的“详情”图标（或学生姓名），右侧会弹出侧滑抽屉显示完整档案。
    - **编辑/删除**：点击末尾操作按钮可更新信息或删除不再需要的记录（**注意**：删除学生将清除其所有历史关联数据）。
- **批量导入**：
    1. 点击右上角“导入学生”按钮。
    2. 下载提供的 Excel 模板。
    3. 按格式填写学生信息并重新上传，系统将自动校验并入库。
- **作业记录**：
    - 在列表的操作栏点击“记录”按钮，可针对特定学生快速录入课程作业完成情况。

### 3.3 班级与教师管理
- **班级管理**：创建并维护班级结构（年级、班名、班主任）。
- **教师管理**：登记在校教师基本信息及所属学科。

### 3.4 成绩管理与学情分析
- **成绩录入**：支持按考试批次、科目快速录入分数。
- **分析报告**：
    - **趋势分析**：查看单个学生或班级的成绩浮动。
    - **排名视图**：自动生成班级排名与年级排名。

## 4. 角色权限说明

系统基于 **RBAC（基于角色的访问控制）**，不同用户看到的菜单和功能有所不同：
- **系统管理员 (Admin)**：拥有最高权限，可进行系统配置、权限分配及日志审计。
- **教务管理员 (Edu Admin)**：主要负责班级、课程和学生的全局维护。
- **教师 (Teacher)**：主要负责所属班级的学生查看、作业记录及本人任教科目的成绩录入。

## 5. 常见问题 (FAQ)

**Q: 为什么我无法看到“系统设置”？**  
A: 该模块仅对系统管理员开放。如需修改功能，请联系技术团队。

**Q: 导入 Excel 时报错怎么办？**  
A: 请确保：1. 使用了系统提供的标准模板；2. 学号没有重复；3. 班级名称与系统内已存在的名称完全匹配。

**Q: 误删学生数据可以恢复吗？**  
A: 数据一旦删除将无法直接通过界面找回。进行删除操作前请务必确认。
