## ✅ 检查清单

### 打开动画（应该看到）
- [ ] 背景遮罩从透明渐变到半透明（300ms）
- [ ] 抽屉从右侧平滑滑入（300ms）

- [ ] 点击遮罩层或关闭按钮

- [ ] 背景遮罩同步淡出（300ms）

## 🐛 常见问题

### 问题：动画不流畅
**解决方案**：
- 检查浏览器是否支持 CSS 动画
- 关闭浏览器开发者工具的动画减速功能
- 检查是否有其他动画冲突

### 问题：内容闪烁
**解决方案**：
- 确保 `content-fade-in` 动画正确加载
- 检查 `globals.css` 是否正确导入

### 问题：遮罩不出现
**解决方案**：
- 检查 z-index 是否正确
- 确保没有其他元素遮挡

## 🎬 动画时间轴可视化

```
时间线：
├─ 0ms    : 触发打开
├─ 0-300ms : 遮罩淡入 + 抽屉滑入
├─ 100ms  : 内容开始淡入
├─ 300ms  : 抽屉完全显示
└─ 400ms  : 内容完全显示

关闭时间线：
├─ 0ms    : 触发关闭
├─ 0-300ms : 遮罩淡出 + 抽屉滑出
└─ 300ms  : 完全关闭
```
